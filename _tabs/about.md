# About

I write down the people, places, and moments that would otherwise disappear.

Not because they are extraordinary,
but because they deserve to be remembered.

Some stories are true.
Some are reconstructed from memory.
Some become fiction before I realize it.

This website is a collection of writings,
rather than a blog.
